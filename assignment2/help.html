HOW TO USE IT

GENERAL KEYS:
HOME - Move to Front page.
RECORD - Move to Record page.
VIEW - Move to view page.

RECORD PAGE HELP:
START - Start recording a run
STOP - Stops recording a run
SAVE - Saves the information of the current run.
CLEAR - Clears current recording and starts it again

VIEW PAGE HELP:
START - Redirect to record page.
CLEAR - Resets the information to 0.
LOAD - Move to recordlist to view info.
DELETE - Move to recordlist to delete info.
